# simple-ng1.5-es5

##A very barebones angularjs app in ES5


## 1. Setup
```bash
npm install
```

## 2. run the development server
```bash
gulp
```

## 3. Build production version
```bash
gulp build
```

## 4. Start webserver without watch task
```bash
gulp server
```

## 5. Start webserver from build folder
```bash
gulp server-build
```
